Introduction

If one has taken calculus, they will know that integration is simply finding the area under a curve. But integration is more than that: applying the different "rules" that can be derived (but are usually memorized) is the hard part. This is called an Integration Bee.

Conclusion

"Recreational" integration is loads of fun. While it may not have much practical use, with symbolic math tools nowadays, it still is helpful for developing problem solving skills.
